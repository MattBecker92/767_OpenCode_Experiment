# Task 1 — Codebase Exploration

## Instructions

Explore the `project/` directory and create a file at `project/SUMMARY.md`.

The file must contain:
- A list of every class in the codebase
- Every method on each class, with a one-line description of what it does
- A list of every module-level function and what it does

## Requirements

1. Only create `project/SUMMARY.md`. Do not modify any existing files.
2. Do not add new dependencies.
3. Run `pytest` to verify that your code passes all tests.
